# 3D Weapon Configurator (Tuning System)

**TUNE3D** — це веб-застосунок для інтерактивного налаштування аксесуарів на 3D-моделях зброї. Проєкт побудований як Full-stack додаток, де користувач може змінювати обвіси, бачити актуальну ціну з бази даних та оформлювати замовлення.

## 📋 Вимоги викладача (реалізовано)
* **GitHub:** Весь вихідний код завантажено в репозиторій.
* **Docker Compose:** Запуск усього стеку однією командою.
* **База даних:** MongoDB працює всередині Docker-контейнера.
* **Міграції:** Реалізовано скрипт для автоматичного наповнення бази даними.
* **Документація:** Детальна інструкція із запуску та описом портів.

---

## 🛠 Технологічний стек
* **Frontend:** Three.js (візуалізація), JavaScript (ES6+), HTML5, CSS3.
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB (запуск через офіційний образ у Docker).
* **Контейнеризація:** Docker & Docker Compose.

---

## 🚀 Як запустити проєкт

Для запуску вам знадобиться лише встановлений **Docker**. Жодних додаткових налаштувань бази чи Node.js на локальній машині не потрібно.

### 1. Запуск контейнерів
У терміналі (в кореневій папці проєкту) виконайте:

docker-compose up --build

### 2. Наповнення бази даними (Міграція)
Коли контейнери запустяться, відкрийте нове вікно терміналу та виконайте команду для завантаження товарів у базу:

docker-compose exec app npm run seed

### 3. Доступ до застосунку
Після запуску перейдіть за адресами:

Конфігуратор (Клієнт): http://localhost:3000

Панель адміністратора: http://localhost:3000/admin.html
